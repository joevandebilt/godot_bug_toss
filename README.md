## Bug Toss
Remember that old Mega Drive version of the lion king where weirdly between 2 levels you played a silly bug toss game where Timon would chuck bugs away and you'd have to dash back and forth as Pumba to catch them?

Decided I should try and remake that in Godot so I could learn the engine basics before a GameJam
